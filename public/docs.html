APIForge Documentation

Create, publish, and monetize REST APIs in minutes. No backend coding required.

Introduction

APIForge is a complete API marketplace platform. You describe your data, we generate a full REST API with endpoints, validation, documentation, and SDKs — instantly.

Quick Start (60 seconds)

# 1. Register
curl -X POST /api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"you@email.com","username":"you","password":"Pass1234!"}'

# 2. Create API instantly
curl -X POST /api/v1/quick/tasks \
  -H "Authorization: Bearer YOUR_TOKEN"

# 3. Use it!
curl /api/v1/live/tasks/tasks \
  -H "X-API-Key: YOUR_KEY"

Authentication

Two auth methods:

MethodHeaderUse For
JWT TokenAuthorization: Bearer TOKENManaging your account/APIs
API KeyX-API-Key: afk_live_xxxConsuming generated APIs

Quick Create

Create an API with a single word:

POST/api/v1/quick/:name Auth

Available: contacts, tasks, notes, bookmarks, inventory, leads, passwords, expenses

AI Generator

Describe what you want in natural language:

POST/api/v1/ai/generate Public
{"description": "API for a gym with members, classes, and trainers"}

Live API Endpoints

Every generated API gets these endpoints automatically:

GET/api/v1/live/:slug/:resource
POST/api/v1/live/:slug/:resource
PUT/api/v1/live/:slug/:resource/:id
DELETE/api/v1/live/:slug/:resource/:id
